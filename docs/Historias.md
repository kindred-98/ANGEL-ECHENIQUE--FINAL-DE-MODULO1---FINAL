5 Historias de Usuario

Como administrador, quiero agregar nuevos libros al sistema,
para mantener actualizado el catálogo de la biblioteca.

Como administrador, quiero registrar usuarios,
para permitirles acceder al servicio de préstamo.

Como usuario, quiero buscar un libro por título o autor,
para encontrar rápidamente el material que necesito.

Como usuario, quiero solicitar el préstamo de un libro disponible,
para poder llevármelo a casa.

Como usuario, quiero devolver un libro prestado,
para que vuelva a estar disponible en el sistema.
